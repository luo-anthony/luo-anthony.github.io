Tournament Manager Discord Bot / Solo Developer

Summer 2020
Github link available on request
The Tournament Manager Discord Bot is a bot that makes hosting and managing Codeword tournaments super easy! It features automatic score tracking and reporting, automated bracket and group generation, (almost) hands-free team and match voice channel management (automatically moves players into game rooms and team rooms), and many more features!
  • Designed and developed a Codewords Tournament Manager Discord Bot w/ proprietary tournament backend (Python, Discord API)
  • Deployed on a live tournament with 8 teams and over 24 players; tested on four different team/group configurations
  • Github repo available on request

3D Connect Four / "Game Engine" Software Developer

Fall 2019 - Art of Engineering Common Project
Github link
3D Connect Four is our take on Connect Four but in 3D! Created as part of a team for Art of Engienering, 3D Connect Four features all new game moves like rotate, flip, and fall on a circular 3D electronic game board! The game also automatically detects when you've won and will display a colorful light show!
  • Developed "game engine" for 3D Connect 4 which drives the game, its fun new game moves, and the game-condition checker (Arduino, C++)
  • Designed, developed, tested, and prototyped game features
  • Github repo
